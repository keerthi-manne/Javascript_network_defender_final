# Network Defender - Color Palette Guide

## Current Theme: Modern Cybersecurity

### Background Colors
- **Primary Background**: `#0d1117` - Deep dark, professional
- **Secondary (Cards)**: `#161b22` - Slightly lighter for panels
- **Tertiary (Elevated)**: `#1f2937` - For raised elements

### Accent Colors (Bright & Vibrant)
- **Primary Accent**: `#00d4ff` - Bright cyan (main UI elements)
- **Secondary Accent**: `#7c3aed` - Purple (highlights)
- **Success**: `#10b981` - Green (source nodes, victories)
- **Warning**: `#f59e0b` - Amber (chokepoints, alerts)
- **Danger**: `#ef4444` - Red (goal nodes, errors)

### Text
- **Primary**: `#f0f6fc` - Bright white
- **Secondary**: `#8b949e` - Muted gray
- **Muted**: `#6e7681` - De-emphasized

### Game Elements
- **Source Node**: Green `#10b981`
- **Goal Node**: Red `#ef4444`
- **Chokepoints**: Amber `#f59e0b` (where towers can be placed)
- **Intermediate Nodes**: Dark gray `#374151`
- **Edges**: Very dark `#1f2937`

### Tower Colors
- **Firewall**: Blue `#50aaff`
- **IDS**: Yellow `#ffc832`
- **Honeypot**: Teal `#00d9ff`

This creates a professional, modern cybersecurity aesthetic with good contrast and visual hierarchy.
